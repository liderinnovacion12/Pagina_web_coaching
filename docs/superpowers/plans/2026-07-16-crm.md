# CRM Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reemplazar el `href: null` de "CRM" por una página real: una única página promocional estática para el CRM externo "GoHighLevel", sin tabla en base de datos ni CRUD admin.

**Architecture:** Un solo server component estático (`app/(estudiante)/crm/page.tsx`) con contenido hardcodeado — mismo patrón que "Curso de Rentas" (`app/(estudiante)/curso-de-rentas/page.tsx`, ya en `main`), pero esta vez usando íconos `lucide-react` (decisión explícita del usuario, a diferencia de Curso de Rentas). Reutiliza los tokens `ink`/`gold`/`mist` y los patrones de tarjeta ya establecidos en `DashboardContent.tsx`.

**Tech Stack:** Next.js 15 (App Router), `lucide-react`, Tailwind (tokens de `tailwind.config.ts`), Vitest + Testing Library.
